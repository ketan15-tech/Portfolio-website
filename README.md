# 🌐 Ketan's Portfolio Website

A modern and responsive personal portfolio website built to showcase my skills, projects, and journey as a final-year Computer Science and Communication Engineering student.

---

##  About the Project

This portfolio represents my work, learning, and growth in software development and AI. It is designed with a clean UI and smooth user experience to attract recruiters and highlight my technical capabilities.

---

##  Features

- Responsive design (mobile-friendly)
- Modern UI with smooth animations
- Custom hero section with profile image
- Projects showcase with descriptions
- Skills and experience sections
- Contact section with social links

---

##   Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Tools:** Git, GitHub, VS Code  
- **Concepts:** Responsive Design, UI/UX, Animations
